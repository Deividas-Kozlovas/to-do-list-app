## Instrukcijos, kaip paleisti

1. **Klonuokite repozitoriją**:

   ```bash
   git clone https://github.com/Deividas-Kozlovas/to-do-list-app
   cd to-do-list-app
   ```

2. **Įdiekite priklausomybes**:

   ```bash
   npm install
   ```

3. **Paleiskite projektą**:
   ```bash
   npm run dev
   ```
